# Todo App

A modern and responsive Todo App built using **TypeScript**, **HTML**, and **CSS**. It allows users to add, delete, and save tasks using the browser's Local Storage, ensuring that tasks remain available even after refreshing or reopening the browser.

# Features

- Add new tasks
- Delete tasks
- Automatic Local Storage support
- Press **Enter** to add a task
- Modern Glassmorphism UI
- Responsive design
- Fast and lightweight

##  Technologies Used

- HTML5
- CSS3
- TypeScript
- Local Storage API
